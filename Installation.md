
### Installation 

Backend:
```bash
cd backend
npm install
npm run dev
```
 Server starts at http://localhost:5000

Frontend (new terminal):
```bash
cd frontend
npm install
npm start
```
 App opens at http://localhost:3000

##  Demo Mode

This project uses mock data for hassle-free demos:
-  No API keys needed
-  No external dependencies
-  Works offline
-  Instant responses
